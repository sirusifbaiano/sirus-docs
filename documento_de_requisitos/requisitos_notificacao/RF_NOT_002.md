## RF_NOT_002: O sistema deve permitir excluir notificações

**Descrição do caso de uso:** O sistema deve permitir que o usuário exclua notificações da sua lista pessoal.

**Ator(es):** Usuário autenticado.

**Prioridade:**   [ ] Essencial       [x] Importante      [ ] Desejável

**Entradas e pré-condições:**
- O usuário deve estar autenticado no sistema.

**Saídas e pós-condição:**
- O sistema remove a notificação da lista do usuário.

**Fluxo de eventos principal:**
- O usuário acessa a paǵina de notificações.
- O usuário seleciona a(s) notificação(ões) que deseja excluir e clica no botão "Apagar".
- O sistema remove a(s) notificação(ões) da lista.

**Fluxos secundários:**


**Requisitos Não-Funcionais (RNF) vinculados:**
- RNF_001: O sistema deve retornar o resultado em menos de 3 segundos.
- RNF_002: O sistema deve ter recursos de acessibilidade.
