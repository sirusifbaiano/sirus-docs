# 📑 Índice de Requisitos Não-Funcionais (RNF)

Esta pasta contém os requisitos não-funcionais gerais do sistema SAMU, que impactam todas as funcionalidades.

| Código      | Requisito | Descrição |
|-------------|-----------|-----------|
| [RNF_001](./RNF_001.md) | Respostas rápidas | O sistema deve retornar o resultado das operações em até 3 segundos sob condições normais de uso. |
| [RNF_002](./RNF_002.md) | Acessibilidade | O sistema deve oferecer recursos de acessibilidade que permitam seu uso por pessoas com diferentes capacidades, seguindo boas práticas de acessibilidade web. |
| [RNF_003](./RNF_003.md) | Identificação de campos obrigatórios | Todos os campos obrigatórios dos formulários do sistema devem ser identificados com um asterisco (*) imediatamente após o seu label. |
